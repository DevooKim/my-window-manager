# My Window Manager

[English](README.md)

비주얼 에디터를 갖춘 macOS 네이티브 윈도우 매니저입니다. 앞쪽 창을 저장해 둔
사각형에 맞춰 스냅하고, 레이아웃을 순환하고, 여러 앱을 모니터 전체에 한 번에
배치합니다 — 모두 전역 단축키로 동작합니다.

## 기능

- **리사이즈 프리셋** — 앞쪽 창을 저장된 사각형에 맞춥니다. 비율과 px를 차원별로
  섞을 수 있습니다(예: "왼쪽 절반 + 높이 200px").
- **사이클** — 단축키 하나로 앞쪽 창을 여러 프리셋 순서대로 넘깁니다.
- **레이아웃** — 여러 앱·여러 모니터를 한 번에 배치합니다. 필요하면 앱을 자동 실행합니다.
- **비주얼 에디터** — 모니터 캔버스에서 드래그로 영역을 그리고 숫자 입력으로 미세 조정합니다.
  단위(비율/px)는 차원별로 지정합니다.
- **전역 단축키** — 각 프리셋·사이클·레이아웃에 원하는 (조합키 + 키) 단축키를 지정합니다.
- **설정 백업** — 전체 프리셋·사이클·레이아웃을 JSON으로 내보내고 가져옵니다.

## 설치

1. [Releases](../../releases)에서 최신 `My-Window-Manager-vX.Y.Z.zip`을 받아 압축을 풉니다.
2. `My Window Manager.app`을 응용 프로그램 폴더로 옮기고 실행합니다.
3. **첫 실행:** 공증(notarize)되지 않은 앱이라 macOS가 차단합니다. 시스템 설정 >
   개인정보 보호 및 보안에서 **"확인 없이 열기"(Open Anyway)** 를 누르고 확인합니다.
4. 안내가 뜨면 손쉬운 사용(Accessibility) 권한을 허용합니다(시스템 설정 >
   개인정보 보호 및 보안 > 손쉬운 사용).

앱은 실행 시와 24시간마다 GitHub Releases에서 업데이트를 확인하고, 새 버전이 있으면
다운로드해 제자리에서 자체 업데이트합니다. 메뉴 막대 → "업데이트 확인..."으로 수동
확인도 가능합니다.

## 요구 사항

- macOS 13 이상
- 손쉬운 사용 권한(시스템 설정 > 개인정보 보호 및 보안 > 손쉬운 사용)

## 소스에서 빌드

```sh
make app          # "dist/My Window Manager.app" 빌드
make run          # 빌드 후 실행
make release      # 배포용 zip 빌드

# 릴리스 흐름:
make bump-minor   # Info.plist 버전 올리고 커밋
make publish      # zip + 태그 + push + GitHub 릴리스 (gh CLI)
```

참고: 로컬 빌드 번들은 자체 서명 인증서("MyWindowManager Dev")로 서명되므로 인증서마다
한 번씩 손쉬운 사용 권한을 부여해야 합니다. 키체인 접근(인증서 지원 > 인증서 생성, 종류:
코드 서명)에서 인증서를 만들면 재빌드해도 권한이 유지됩니다. 다른 이름을 쓰려면
`WM_SIGN_IDENTITY`를 설정하세요. 수동 검증 체크리스트는
[docs/smoke-test.md](docs/smoke-test.md)를 참고하세요.

## 설정

`~/Library/Application Support/MyWindowManager/config.json`
